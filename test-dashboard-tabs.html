Prueba del Sistema de Pestañas del Dashboard

Instrucciones de Prueba

Esta página permite probar el sistema de pestañas con deep-linking del dashboard. Los enlaces a continuación cargarán el dashboard con diferentes pestañas activas:

  • Mis Proyectos: Muestra los proyectos creados por el usuario
  • Mis Aportes: Muestra el historial de donaciones realizadas
  • Favoritos: Muestra los proyectos marcados como favoritos

El sistema debería:

  1. Cargar la pestaña correcta según el hash en la URL
  2. Actualizar el banner dinámicamente
  3. Permitir navegación entre pestañas sin recargar la página
  4. Mantener el estado correcto al usar los botones de navegación del navegador

Verificación Manual

Después de hacer clic en los enlaces anteriores, verifica:

  • ✅ La pestaña correcta está activa (resaltada visualmente)
  • ✅ El contenido correspondiente se muestra
  • ✅ El banner superior se actualiza con el texto correcto
  • ✅ La URL en el iframe muestra el hash correcto
  • ✅ Los botones de las pestañas funcionan correctamente
  • ✅ Los botones de acción dentro de cada pestaña responden al clic
